# Sistema de Banco de Dados – Academia

Banco de dados relacional desenvolvido em MySQL para gerenciamento de uma academia.

## Funcionalidades
- Cadastro de alunos e professores
- Controle de planos e pagamentos
- Gerenciamento de treinos e aulas
- Relacionamentos com chaves estrangeiras

## Tecnologias
- MySQL
- SQL padrão

## Estrutura
- alunos
- professores
- planos
- treinos
- aulas
- pagamentos

## Como executar
1. Execute `schema.sql`
2. Execute `seed.sql`
3. Consulte os dados com SELECT

## Objetivo
Projeto acadêmico e de portfólio para demonstrar modelagem de banco de dados relacional.
